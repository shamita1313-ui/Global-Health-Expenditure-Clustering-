# Global-Health-Expenditure-Clustering-
Analyzes global health spending using the IHME Health Spending 1995–2021 dataset in Python/Colab. Performs country clustering, PCA, and interactive visuals. Results are presented with code and screenshots for quick insights and transparent, reproducible portfolio documentation

## Overview

- **Goal:** Cluster countries by their health expenditure patterns and visualize results with clear, interactive charts.
- **Tech stack:** Python, pandas, matplotlib, seaborn, plotly, Google Colab
- **Source data:** IHME Global Health Spending

---

## Repository Structure

| Folder/File         | Contents                                              |
|---------------------|------------------------------------------------------|
| `powerbi1.ipynb`    | Main Google Colab notebook with complete code        |
| `data/`             | Raw and processed CSV files from IHME and analysis   |
| `screenshots/`      | All final visual outputs and table images            |
| `README.md`         | Project documentation (this file)                    |

---

## Key Features

- Country-level health expenditure clustering using KMeans
- Principal Component Analysis (PCA) for visualization
- Interactive and static charts: bar, map, and scatter plots
- Screenshots for main results (see `screenshots/`)
- Transparent, reproducible workflow in Colab notebook
